# Knowledge Update — KNOW-0005 / EXP-0008 / RUN-0001

The standard Koide formula Q = 2/3 does not hold for quark mass triplets
under PDG 2024 masses. Up sector: Q = 0.848981 (159.2σ above 2/3). Down sector: Q = 0.731497 (8.8σ above 2/3).
Phase-modified formula Q(δ) also cannot reach 2/3 for either sector.
See docs/notes/koide-quark-cascade.md.
