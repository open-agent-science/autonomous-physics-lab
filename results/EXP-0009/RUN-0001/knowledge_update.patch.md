# Knowledge Update — EXP-0009 / RUN-0001

An analytic shell correction to the Bethe-Weizsäcker formula based on
per-magic-number Gaussian bumps (multi_gauss_NZ_shared_sigma) achieves
28.8% test-set RMS improvement on a curated AME2020 subset.

Key finding: distance-to-nearest-magic-number models give <1% improvement.
Individual amplitudes per magic number are required for significant improvement.
See knowledge/nuclear_physics/binding_energies.yaml for the dataset.
