APL is an open-source, verification-first engine for physics hypotheses.

Current benchmark slices:

- pendulum formula discovery
- damped oscillator regime verification

The core idea is public scientific memory:

`hypotheses -> experiments -> results -> claims -> knowledge`

Current repo:

[gladunrv/autonomous-physics-lab](https://github.com/gladunrv/autonomous-physics-lab)

What is included now:

- reproducible benchmark runs
- run-based result artifacts
- repository validation
- contributor workflow
- claim-promotion policy

What is not included yet:

- literature ingestion
- dashboard
- multi-agent runtime
- theory graph
