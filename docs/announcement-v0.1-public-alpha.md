# Public Alpha Announcement Draft

Historical filename note: this file began as a `v0.1` draft, but the current
announcement posture is the cautious `v0.2-public-alpha` opening. Do not use
this text until the release gates in `docs/public-release-gates.md` are
satisfied and the maintainer explicitly chooses to open the repository.

APL is an open agent network for reproducible physics research.

The launch story is not "AI discovered new physics." The launch story is that
human-owned coding agents can contribute to shared scientific memory when the
repository forces them through tasks, source gates, deterministic checks,
limitations, PR review, negative results, and replay.

Current public-safe evidence hooks:

- Exoplanet Mass-Radius has a pinned benchmark surface and a null-baseline
  control panel; its current compact-radius signal is preserved as
  benchmark-diagnostic memory, not a planet-composition or habitability claim.
- Materials Property Residuals has `MD-0001`, the first source-pinned reusable
  dataset candidate with replayed baseline evidence and explicit publication
  blockers.
- Nuclear Mass Surface has a frozen baseline, prediction registry discipline,
  and useful negative/control memory; reveal scoring remains blocked until
  future source-grade no-peek data exists.
- Quantum Size Effects and Atomic-Clock Residuals show the fresh-data gate:
  source artifacts, checksums, provenance, row admissibility, covariance, and
  blockers before modeling.
- Pendulum, Dimensional Analysis, Particle Mass Relations, and Anharmonic
  Oscillator remain the quality-floor benchmarks and falsification tracks.

The core loop is public scientific memory:

`mission -> task -> evidence -> deterministic check -> verdict -> review -> memory`

Current repo:

[open-agent-science/autonomous-physics-lab](https://github.com/open-agent-science/autonomous-physics-lab)

What is included now:

- reproducible benchmark and falsification results;
- source-pinned dataset and source-artifact surfaces;
- agent-published and agent-validated review-tier vocabulary;
- prediction registry discipline for future reveal data;
- branch/PR/review/closeout workflow for external agents;
- no automatic claim or knowledge promotion.

What is not included yet:

- discovery-level physics claims;
- automatic theory generation;
- open-ended formula search;
- public dataset DOI releases;
- source ingestion that bypasses license, checksum, or provenance review.
