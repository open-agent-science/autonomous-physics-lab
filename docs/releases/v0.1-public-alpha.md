# v0.1-public-alpha

Prepared release notes for a future public alpha of Autonomous Physics Lab.

Historical note: this draft reflects an earlier public-alpha cut. It is not
the current `v0.2` campaign snapshot and should be read as a historical
release-preparation artifact rather than the current repository status page.

## Included

- `EXP-0001`: Pendulum Formula Discovery
- `EXP-0002`: Damped Oscillator Regime Verification
- run-based result artifacts under `results/<experiment>/<run>/`
- public scientific memory directories for:
  - hypotheses
  - experiments
  - results
  - claims
  - knowledge
  - tasks
- claim promotion policy
- contributor workflow and GitHub templates
- artifact input-hash drift validation
- immutable canonical run-input snapshots
- CI validation on Python `3.11` and `3.12`

## Scientific Scope

This release is intentionally narrow.

It demonstrates two verification-backed benchmark slices:

1. pendulum approximation benchmarking with range-aware verdicts;
2. damped-oscillator regime verification against exact linear solutions.

Later campaign results, including dimensional-analysis validation and Koide
follow-up falsifications, are documented elsewhere and were not part of this
earlier draft.

It does not claim:

- general physical theory discovery;
- broad benchmark coverage;
- autonomous scientific review without human judgment.

## Current Strengths

- deterministic workflows;
- reproducible result artifacts;
- verification summaries beyond simple fit quality;
- repository-wide validation and referential integrity checks;
- contributor-facing public-memory structure.

## Current Limits

- pendulum evidence remains range-limited;
- theory-aware pendulum candidates improve separatrix behavior but do not become exact;
- claim promotion remains a maintainer review step;
- only two benchmark slices are implemented.

## Not Included Yet

- literature ingestion
- dashboard
- multi-agent runtime
- theory graph
- third benchmark slice

## Next Focus

- strengthen claim/evidence semantics further where useful;
- preserve release discipline for canonical artifacts;
- plan the next benchmark only after the current public-alpha state remains stable.
