# TEC-001: Light Clock Scenario — Assumption Formalization

## Scenario

A photon bounces vertically between two mirrors separated by height h in rest
frame S. Frame S' moves at velocity v relative to S along the horizontal axis.

## Explicit Assumptions

1. **Inertial frames:** S and S' are both inertial. No acceleration, rotation,
   or gravitational field. The scenario is strictly special relativity; general
   relativistic effects are excluded by assumption.

2. **Constant speed of light:** c is constant and isotropic in all inertial
   frames (Einstein's second postulate). No alternative synchronisation
   convention is used.

3. **Mirror geometry:** mirrors are perpendicular to the vertical axis,
   separated by rest-frame distance h. Mirror length is not specified and is
   assumed irrelevant for the period calculation.

4. **Single-photon model:** the clock pulse is treated as a single photon.
   Wave effects, beam divergence, and finite mirror size are neglected.

5. **No mechanical effects:** mirror mass, photon recoil on the mirror, and
   mirror deformation are neglected.

6. **Einstein synchronisation:** clocks in S and S' are synchronised using
   the Einstein convention. No alternative convention is used.

## Undefined Region

v ≥ c is undefined. The time-dilation factor γ = 1/√(1−v²/c²) diverges as
v→c and is imaginary for v>c. This is a hard domain boundary, not a limit
to be extrapolated past.

## Limitation Statement

Planning note only. No executable benchmark exists. These assumptions apply
to the simplified SR light-clock model only. Empirical validation and
general-relativistic corrections are out of scope.
