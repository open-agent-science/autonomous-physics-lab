## Scientific content (frozen main-sequence holdout, dex)

| baseline | holdout MAE |
|---|---|

| textbook piecewise α=4.0 | 0.137608 |
| textbook single α=3.5 | 0.184954 |
| per-mass-band median null | 0.331817 |

- Single α=3.5 beats the null but the fitted/piecewise gaps (0.065 / 0.047 dex)
  exceed the 0.04 dex split-noise reference ⇒ **single α=3.5 inadequate as sole baseline**.

  (0.146863) exceeds all shuffled; deterministic replay reproduces all numbers (tol 1e-6).
