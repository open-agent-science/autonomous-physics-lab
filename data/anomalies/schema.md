# Anomaly Registry Schema

This directory currently contains only the planning schema for future anomaly
registry entries:

- [`anomaly.schema.json`](./anomaly.schema.json)

`TASK-0308` does not add anomaly rows, run joint-likelihood fits, promote
claims, or move any WATCHLIST topic into active evaluation. The schema defines
the admissibility and holdout metadata a later maintainer-approved task must
use before any real anomaly entry can be considered.

Use the schema with:

- [Anomaly Registry Campaign](../../docs/campaigns/anomaly-registry.md)
- [Anomaly Registry Admissibility](../../docs/notes/anomaly-registry-admissibility.md)
