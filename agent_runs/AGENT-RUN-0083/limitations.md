# Limitations

- Two materials only (InP, ZnSe); a bounded two-material transfer benchmark, NOT a universal size law, quantum-dot design law, or material recommendation.
- Confinement-term framing (E1s - E_bulk) with explicit cited bulk gaps (InP 1.34 eV, ZnSe 2.70 eV) and an equal-volume edge->diameter conversion.
- Direct-size rows only (InP TEM edge length, ZnSe SAXS diameter); the calibration-derived Yu CdSe / Moreels PbS sets are excluded.
- Six InP rows and ten ZnSe rows; one source and one morphology per material.
- Sandbox evidence only; no RESULT, PRED, CLAIM, or KNOWLEDGE artifact is created and no claim is promoted.
