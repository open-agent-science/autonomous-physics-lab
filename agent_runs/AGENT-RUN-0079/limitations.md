# Limitations

- Single pinned COBE/FIRAS absolute monopole product (43 rows, one spectral axis).
- One separately pinned reference temperature (`Fixsen 2009, ApJ 707, 916`); the FIRAS-recalibrated value is reserved for sensitivity only.
- The reference temperature is FIRAS-derived, so the audit is a blackbody self-consistency check, not independent empirical validation of Wien displacement.
- Raw-bin peak precision is limited by FIRAS bin spacing; the interpolation is a fixed sampling-resolution diagnostic, not a fit.
- Sandbox-only: no canonical RESULT/CLAIM/KNOW/PRED artifact is created.
- Not a blackbody-universality, textbook-falsification, proof, or discovery statement.
