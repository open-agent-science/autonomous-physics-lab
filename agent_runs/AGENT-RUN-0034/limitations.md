# Limitations

- This is a retrospective slice audit over a committed snapshot and a frozen
  baseline artifact.
- The audit does not refit Chen-Kipping parameters or tune a replacement
  baseline.
- Slice labels are residual diagnostics only; they do not imply a planet
  composition law.
- Host-star and uncertainty slices are affected by catalog selection and
  follow-up history.
- Minimum-mass rows remain outside headline metrics because only two
  post-filter rows are eligible and `M sin i` is not true mass.
- No habitability, biosignature, target-priority, prediction registry, claim,
  knowledge, or canonical result output is authorized.
