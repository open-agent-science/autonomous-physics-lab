# Exoplanet Mass-Radius Predictions

This directory stores prospective, pre-reveal predictions for the Exoplanet
Mass-Radius campaign.

## Units And Quantities

- Primary campaign observables use Earth units for mass and radius in the
  pinned exoplanet source artifacts.
- The first registered entry uses a categorical protocol-outcome quantity,
  not a numeric planet measurement: `future_second_snapshot_reveal_outcome`.

## Accepted Source Classes

- First-snapshot frozen state:
  `data/exoplanets/exo-0001-pscomppars-snapshot.yaml`.
- Future comparison source class:
  maintainer-approved NASA Exoplanet Archive PSCompPars second snapshot using
  the committed ADQL query contract or a separately merged pre-reveal
  amendment.

## Reveal Cadence

Reveal requires a separate maintainer-approved task that pins the exact source
URL, acquisition actor, retrieval timestamp, raw and normalized artifacts,
row counts, field list, checksums, and no-peek attestation before scoring.
Registration alone is not a result, claim, or knowledge artifact.
