# Hypothesis Register

Lifecycle: **legacy pilot registry**.

This directory preserves the early `HRE-*` hypothesis-register pilot. It is no
longer the default path for new hypotheses.

Use these current paths instead:

- `hypothesis_proposals/` for campaign-scoped unreviewed hypothesis ideas;
- `hypotheses/` for reviewed hypothesis artifacts;
- `results/` for reproducible experiment outputs;
- `claims/` or `knowledge/` only after the required promotion and maintainer
  review gates.

Do not add new `HRE-*` entries unless a maintainer explicitly reactivates this
surface. Existing entries remain useful historical memory and should not be
deleted casually.
